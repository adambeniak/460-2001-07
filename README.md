# 460-2001/07 – Algorithms I (ALG I)

## Course Objectives Expressed as Achieved Skills and Competencies
To introduce students to problem-solving techniques using algorithms. Upon completing the course, the student will be able to:
- Define and describe selected problem-solving techniques using algorithms.
- Demonstrate these techniques on sample problems.
- Utilize these techniques to solve other problems.
- Work with combinations of multiple techniques together.

## Course Annotation
This course is one of the introductory programming courses. The course aims to familiarize students with techniques and strategies for problem-solving using algorithms. The algorithms and data structures covered will be demonstrated in the C++ language. Students are guided to analyze algorithmic problems and synthesize solutions from smaller components.

## Forms of Assessment and Additional Student Requirements
### Conditions for obtaining the credit:
- Implementation and defense of a project.
- Programming simple applications during practical exercises.
- Attendance at practical exercises.
- Completion of several homework assignments.

## Additional Student Requirements
Students are expected to have a basic knowledge of the C++ language and high school mathematics.

## Course Syllabus
1. Algorithm. Strategies for problem-solving using algorithms. Significant types of solved problems.
2. Analysis of algorithmic complexity.
3. Brute force problem-solving strategy. Selection sort, bubble sort. Sequential search. Convex hull of a set of points. Finding the closest pair of points.
4. Exhaustive search (exhaustive search/complete exploration) strategy. Traveling salesman problem. Knapsack problem. Graph traversals.
5. Decrease-and-conquer problem-solving strategy. Insertion sort. Generating permutations and subsets. Binary search. Finding the median. Interpolation search. Searching and inserting into a binary search tree.
6. Divide-and-conquer problem-solving strategy. QuickSort. MergeSort. Convex hull of a set of points. Finding the closest pair of points.

## Laboratory Exercises Content
1. Complexity analysis of iterative algorithms.
2. Complexity analysis of recursive algorithms.
3. Implementation of the convex hull of a set of points. Implementation of finding the closest pair of points.
4. Traveling salesman problem - experiments with exhaustive search.
5. Utilization of graph traversals.
6. Implementation of algorithms for generating permutations and subsets.
7. Experiments with binary search and interpolation search. Finding the median.
8. Implementation of a binary search tree.
9. Implementation of solutions using the divide-and-conquer strategy.
